# Zeus
A BEM flavoured, mobile first flexbox grid framework.

<img src="https://cloud.githubusercontent.com/assets/2805249/15032804/00168fa8-12a8-11e6-8a19-4c515f7f359c.jpg" style="width: 100%;">

Check out the official web page at [http://enzyme.github.io/zeus/](http://enzyme.github.io/zeus/)

## Installation

#### Using a CDN
Minified

`https://cdn.rawgit.com/enzyme/zeus/master/bin/zeus.min.css`

Development

`https://rawgit.com/enzyme/zeus/master/bin/zeus.css`

#### NPM
`$ npm install zeus-grid`

#### Bower
`$ bower install zeus-grid`

## Example usage
```html
<div class="Layout">
    <div class="Layout__row">
        <div class="Layout__column Layout__column--xs-3">
            <p>Tellus Quam Purus Risus Etiam</p>
        </div>
        <div class="Layout__column Layout__column--xs-3">
            <p>Risus Aenean Ridiculus Consectetur Bibendum</p>
        </div>
        <div class="Layout__column Layout__column--xs-3">
            <p>Risus Porta Ultricies</p>
        </div>
        <div class="Layout__column Layout__column--xs-3">
            <p>Sit Ultricies Euismod Inceptos</p>
        </div>
    </div>
</div>
```
