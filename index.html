Responsive

Designed for mobile first development and inspired by Bootstrap, the grid is responsive to a variety of different viewport sizes.

Go ahead and try resizing your browser window!

Flex

Each column takes up an equal amount of space on the grid across all viewports, or a specific percentage using the size modifiers.

.Layout__column or

.Layout__column--*

XS <544px

For tiny devices like mobile phones. These columns are the starting point for any grid which are then overriden by larger viewport modifiers.

.Layout__column--xs-*

SM >=544px

For handheld devices like large phones or small tablets. These columns will be collapsed until the minimum viewport size has been reached.

.Layout__column--sm-*

MD >=768px

For larger handheld devices such as tablets. These columns will be collapsed until the minimum viewport size has been reached.

.Layout__column--md-*

LG >=992px

For devices such as laptops or smaller screen desktops. These columns will be collapsed until the minimum viewport size has been reached.

.Layout__column--lg-*

XL >=1200px

For your large desktop and general purpose displays. These columns will be collapsed until the minimum viewport size has been reached.

.Layout__column--xl-*

Customizable

Need a specific container size? Want more gutter space? Need more or less columns? We've got you covered!

Simply by using the SASS version of Zeus, you can override the variables $container-width $gutter-width and $column-count respectively.
